## Blockchain Standards

A curated, practical set of standards and best practices for designing, building, operating, and securing blockchain networks and applications.

### Blockchain Standards Repositories

- `BlockchainStandards/` — Source Markdown for each standard (one file per topic)
- `website/` — Hugo-based static website for publishing the standards
- `examples/` — Example materials accompanying the standards

### Topics covered

Standards are organized by domain, including:

1. Hardware and infrastructure
2. Communications and networking
3. Development practices
4. DevOps automation
5. Build-from-source
6. Binary installation packages
7. Docker containers
8. Kubernetes
9. Upgrades and lifecycle management
10. Security practices
11. Monitoring and observability
12. Resource efficiency and sustainability

Authoritative Markdown lives under `BlockchainStandards/`. The website contains rendered and occasionally adapted versions under `website/content/standards/`.
